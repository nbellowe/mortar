# Operator Docs

This directory is for documentation aimed at the person installing, configuring, and running Mortar.

Write these docs for technical readers:

- explain config fields and environment variables
- document supported plugin combinations and compatibility notes
- include deployment, upgrade, and troubleshooting guidance when relevant

Examples:

- Installing Mortar with Docker
- Configuring the Jellyseerr plugin
- Upgrading between Mortar releases
