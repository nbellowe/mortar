# Code of Conduct

Mortar aims to be a respectful, low-ego project for users, operators, and contributors.

## Expected behavior

- Be kind and constructive
- Assume good intent, but be clear and honest
- Focus feedback on ideas, code, docs, and decisions rather than people
- Help keep the project approachable for newcomers

## Unacceptable behavior

- Harassment, intimidation, or personal attacks
- Discriminatory language or conduct
- Deliberate hostility, dogpiling, or bad-faith disruption
- Sharing private or sensitive information without permission

## Scope

This applies to project discussions, issues, pull requests, reviews, and other project spaces.

## Enforcement

The maintainer may moderate, edit, or remove content that undermines a safe and collaborative project environment.
