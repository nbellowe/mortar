# Using Mortar

Mortar is your household's single app for finding and requesting movies, TV shows, audiobooks, and other media.

You don't need to know anything about the services running behind it.

## What you can do

| Task | Where to go |
|---|---|
| Search for something and request it | [Requesting media](./requesting) |
| Browse what's already available | [Browsing your library](./browsing) |
| Check if a download is in progress | The home screen shows recent activity |
| See what was added recently | The home screen shows recently added titles |

## Signing in

Open Mortar in your browser (your operator will share the URL), then sign in with the username and password they gave you.

If you forget your password, ask whoever runs the server to reset it — there is no self-service password reset.
