# App User Docs

This directory is for documentation aimed at people using Mortar itself.

Write these docs in plain language:

- focus on tasks and outcomes
- avoid homelab jargon where possible
- do not require readers to know what Sonarr, Radarr, or SABnzbd are

Examples:

- How to request a movie
- How to check whether something is downloading
- How to browse what is already available
