# Requesting media

## How to request something

1. Go to **Search** in the navigation.
2. Type the name of a movie, TV show, or audiobook.
3. Results appear with a status badge:
   - **Available** — already in the library, you can watch or listen now
   - **Requested** — someone already requested it, it will be added when ready
   - **Request** — not yet available; tap to request it
4. Tap **Request** to submit. You'll see the badge change to **Requested**.

That's it. Mortar passes your request to the right service and it will be added when it's ready.

## Checking on a request

Mortar doesn't send notifications yet. To check whether something was added, search for it again — the badge will show **Available** once it's in the library.

## Something went wrong

If you tap Request and see an error, the underlying service may be unreachable. Let your operator know; there's nothing more you can do from the app.
